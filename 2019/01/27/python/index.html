python学习

Posted on 2019-01-27 | Edited on 2019-02-20

python是最近最新兴起的一门计算机语言相较于其他的语言更加精炼更加简单，但深入学习python需要很硬的功底，功底很重要。

以下是python的学习笔记：

  1. python的存储类型总共分为几大类：变量，字符串，列表，元组，字典，类。

  2. python的语句类型与c语言相似if语句，while循环，if循环，不同的是python的语法更加精炼没有太多的限制，但要求更多的是代码缩进，逻辑思考的清晰度，更要有整体的想法思路。

    基础章,python赋值储存的方式。

    1. python的段行输出有两种输出方式 print 与echo，都能输出相应变量的值。

      print(“hello world”)
      echo(“hello world”)

    2. 变量的赋值

    • 加、减、乘、除和取余都是标准运算符。Python 有两种除法运算符，单斜杠用作传统除法， 双斜杠用作浮点除法。传统除法是指如果两个操作数都是整数的话， 它将执行是地板除，而浮 点除法是真正的除法，不管操作数是什么类型，浮点除法总是执行真正的除法。
      x = y　　　　　　　　　　 x 等于y
      x < y　　　　　　 　 x 小于y
      x > y　　　　　　 　 x 大于y
      x >= y　　　　　　 x 大于等于y
      x <= y　　　　　　 　x 小于等于y
      x != y　　　　　　　　　 x 不等于y
      x is y　　　　　　　　　 x 和y是同一个对象
      x is not y　　　　　　　 x 和y是不同的对象
      x in y　　　　　　　　　 x 是y容器(例如，序列)的成员
      x not in y　　　　　　　 x 不是y容器(例如，序列)的成员

    • 获取用户输入的值

    交互器会识别用户输入的值并将其中的值赋给变量或者打出。

    1. 字符串的赋值
      字符串的赋值类型没有像c语言麻烦与数字赋值一样都是在基本的赋值语句就能够完成

  3. 列表

  • 列表如c语言的数组相近储存多个单元的信息，并一一对应位置。

  • 从一位0开始计数，列表还有切片将其分开表达，其后数组中的值有相应的法则进行改变。

  • 而在元组中却无法将其改变，初始定值之后元组的值将保持不变，除非将其全部进行二次赋值

  • python的列表还有很多能够改变的方式：
    | 操作 | 方法 |
    | ————- |:——————-:|
    | 删除个别列表的元素 | del x[?/]
    | 表示列表长度 | len(x) |
    | 返回最大值 | max(x) |
    | 返回最小值 | mim(x) |
    | 转化元组为列表 | list(x) |
    | 添加元素到末尾 | x.append(??) |
    | 统计元素在列表出现的次数 | x.count(??) |
    | 插入一个对象至列表 | x.insert(插入位置的下标，值) |
    | 弹出列表某一位的值并表达 | x.pop(x) |

  1. 字典
    字典是 Python 中的映射数据类型，工作原理类似 Perl 中的关联数组或者哈希表，由键值(key-value)对构成。几乎所有类型的 Python 对象都可以用作键，不过一般还是以数字或者 字符串为常用用。
    值可以是任意类型的 Python 对象，字典元素用大括号({ })包裹。

    zidian={‘x’:2,’y’:1,’z’:3}
    储存单元信息也可以随时更改。
    zidian[‘x’]=3

  • 字典的基本操作
    | 方法 | 操作 |
    | ————- |:——————-:|
    | len(d) | 返回 d 中项(键-值对)的数量 |
    | d[k] | 返回关联到键 k 上的值 |
    | d[k]=v | 将值 v 关联到键k上 |
    | del d[k] | 删除键为 k 的项 |
    字典也有许多内建函数
    1 clear 方法清除字典中所有的项。

发现其中的d值已经被清空。
2 copy方法返回一个具有相同键-值对的新字典，储存在一个新的内存中。

3 fromkeys方法使用给定的键建立新的字典，每个键都对应一个默认的值None

d={}
d.fromkeys([“name”, “age”])
如若不想使用None作为默认可以提供或者更改两种方法
4 items 方法将字典所有的项以列表方式返回，列表中的每一项都表示为(键, 值)对的形式。但 是项在返回时并没有遵循特定的次序。
x={‘name’:’tom’,’age’:’18’}
x.items()
返回所有的值键对，但顺序却是散乱的。
5 pop 方法用来获得对应于给定键的值，然后将这个键-值对从字典中移除。

返回得到一个值但字典该位置的值却消失

  1. 字符串的操作
    ·find find 方法可以在一个较长的字符串中查找子串。它返回子串所在位置的最左端索引。如果没 有找到则返回 -1 。

    “With a moo-moo here, and a moo-moo there”.find(“moo”)
    查找结果返回7；
    ·join 方法是非常重要的字符串方法，它是 split 方法的逆方法，用来连接序列中的元素：

· lower 方法返回字符串的小写字母版。

“Trondheim Hammer Dance”.lower()
·replace方法返回某字符串的所有匹配项均被替换之后得到字符串。
“This is a test”.replace(“is”,”eez”)
·split这是一个非常重要的字符串方法，它是 join 的逆方法，用来将字符串分隔成序列。
“1+2+3+4+5”.split(“+”)
返回一个序列[‘1’, ‘2’, ‘3’, ‘4’, ‘5’]。
如果不提供任何分隔符，程序会把所有空格作为分隔符(空格、制表、换行等)。
·strip 方法返回去除两侧(不包括内部)空格的字符串：
“ internal whitespace is kept “.strip()
返回生成的结果’internal whitespace is kept’

python语法

python中有许多语法上与c语言语法相似if-elif-else，while，for.在其中又有一些稍稍的变动
中间出去省略的对{}的依赖使代码变得精简客观。

  1. if-elif-else语句相较于c中没有太大变化，但在else if 时python却将其变为elif

  2. while 语句与c来比较没有太大的差距 但所有的语句段都需要在语句后加上‘：’
    如：想要打印1~100个数字用while实现。
